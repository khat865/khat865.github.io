A Visual Detection Method for Surface Defects on Chips Based on The Improved YOLO11
  • Evaluated mainstream object detection models (e.g., YOLO series, Fast-RCNN) and selected YOLO11 as the baseline
  • Incorporates a heatmap structure to enhance focus on key regions, improving detection of small and blurry targets
  • Designs an adaptive multi-feature fusion module using parametric linear combination and soft clustering to generate an optimized fourth-channel feature, strengthening discrimination of low-contrast defects
Skeleton-Based Action Recognition
  • Reviews and groups skeleton-based action recognition methods into four representation-based categories, analyzing how each influences spatio-temporal modeling selection
  • Benchmarks 19 methods on the Anubis dataset (e.g., STGCN, DeGCN, LAGCN), revealing model rankings and correlations between performance and input representation types
  • Shows that multi-representation fusion only benefits specific actions, uncovers action-representation dependencies, and highlights limitations in fine-grained and interaction recognition
